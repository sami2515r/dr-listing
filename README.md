# Dr Listing System

A healthcare doctor and hospital listing platform built using PHP, MySQL, JavaScript, HTML, and CSS.

## Features

- Doctor registration/login
- Admin panel
- Doctor approval system
- Hospital management
- Doctor-hospital request workflow
- Reviews and ratings
- Doctor schedules
- Search and filtering
- Dashboard management

## Technologies Used

- PHP
- MySQL
- JavaScript
- HTML/CSS
- Bootstrap
- WAMP Server

## Setup Instructions

1. Clone repository
2. Move project to WAMP `www` folder
3. Import database from:

```txt
database/listing_dir.sql
```

4. Update database credentials in:

```txt
config/db.php
```

5. Start Apache and MySQL
6. Open:

```txt
http://localhost/dr_listing
```
